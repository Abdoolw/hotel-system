# Hotel Management System (Laravel 11)

A comprehensive Hotel Management System built with Laravel 11. This project provides a robust platform for managing hotel operations, including room bookings, guest management, staff roles, and permissions.

## Features

- User authentication (Admin, Staff, Guest)
- Role-based access control (using spatie/laravel-permission)
- Room management (CRUD)
- Booking management (CRUD)
- Guest management
- Staff management
- Permissions and roles management
- Dashboard with hotel statistics
- Responsive UI with Blade templates

## Getting Started

Follow these steps to set up and run the project:

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/hotel-system.git
cd hotel-system
```
